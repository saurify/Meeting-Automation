This program automates the process of joining meetings on microsoft teams and google meets in chrome browser.

Instructions to use:
Step 1. Paste you meeting links in "Calender.xls" file, as per your schedule
Step 2. Run login.py file 
Step 3. Login into your google and microsoft teams account in prompted browser
Step 4. Close browser and the python script running in background from step 2.
Step 5. Run control.py file

Notes:
1. Do not interact with browser after running control.py
2. Find chromedrivers for your browser version, you can check your browser version in settings, in this repo I have attached driver for version 87
3. This schedules meetings using your local machine time, so make sure your date and time settings are set correctly
4. This is for education purposes only.
